# AI Data Startup Lab

Data Startup Simulation 강의를 따라 학생들이 크라우드펀딩 경쟁 제품을 찾고, 선택한 제품을 바탕으로 창업 아이디어를 고도화하는 웹 실습 도구입니다.

학생은 관심사를 입력해 와디즈, 텀블벅, 오마이컴퍼니, 펀딩포유 등 국내 크라우드펀딩 제품 후보를 추천받고, 상세 페이지를 직접 확인한 뒤 최종 경쟁 제품을 선택합니다. AI는 후보 탐색, 분석 질문, 카드 생성, 결과물 정리를 도와주는 조력자로만 동작합니다.

## 실행

검색 기능은 서버에서 OpenAI API를 호출합니다. 학생 브라우저에는 API 키가 노출되지 않습니다.

1. PowerShell에서 API 키 설정

```powershell
$env:OPENAI_API_KEY="sk-..."
```

2. 로컬 서버 실행

```powershell
.\run_app.ps1
```

3. 브라우저에서 접속

```text
http://127.0.0.1:5173
```

## Vercel 배포

이 프로젝트는 Vercel에서 정적 페이지와 서버리스 API로 동작합니다.

필수 설정:

- Vercel 프로젝트 Settings -> Environment Variables에서 `OPENAI_API_KEY` 추가
- Production, Preview, Development 환경에 모두 적용 권장
- 환경변수 저장 후 Redeploy 실행

배포 후 주요 경로:

- `/` : 실습 웹앱
- `/api/status` : API 키 연결 상태 확인
- `/api/recommend` : 국내 크라우드펀딩 제품 추천
- `/api/image` : 상세 페이지 대표 이미지 프록시

## 현재 기능

- 8차시 워크시트 흐름 기반 실습 진행
- 관심사 기반 국내 크라우드펀딩 제품 후보 5개 추천
- 추천 후보의 상세 페이지 링크 제공
- 상세 페이지 대표 상품 이미지 자동 추출
- 선택한 경쟁 제품 기반 아이템 카드 자동 생성
- 생성 카드 PNG 다운로드
- 선택한 경쟁 제품을 이후 차시 분석 기준으로 자동 연결
- 변환 키워드 선택
- 프로젝트명과 작성 내용 자동 저장
- 차시별 교육 목적, 완성 기준, 작성 필드 제공
- AI 제안을 학생이 직접 적용하거나 반려하는 구조
- 최종 사업계획서 요약과 발표용 슬라이드 구성 미리보기
- 보고서 텍스트 다운로드
- 프로젝트 JSON 저장

## 설계 원칙

- AI가 정답을 확정하지 않고 학생의 선택을 돕습니다.
- 경쟁 제품은 학생이 상세 페이지를 직접 확인한 뒤 최종 선택합니다.
- 오래된 예시 카드는 데이터로 사용하지 않고, 카드 레이아웃의 참고 형식으로만 봅니다.
- 교육 목적이 드러나도록 시장 탐색, 문제 정의, 고객 분석, 고도화, 사업화 흐름을 유지합니다.
